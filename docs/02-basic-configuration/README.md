# Базовая настройка Foreman

Начальная настройка Foreman после установки.

## Первый вход

1. Откройте веб-интерфейс: `https://192.168.0.209/`
2. Логин: `admin`
3. Пароль: (получите командой `foreman-rake permissions:reset`)

## Создание организации и локации

1. **Организация**:
   - Перейдите в **Настройка → Организации**
   - Создайте или используйте "Default Organization"

2. **Локация**:
   - Перейдите в **Настройка → Локации**
   - Создайте или используйте "Default Location"

## Настройка операционной системы

1. Перейдите в **Настройка → Операционные системы**
2. Создайте или выберите:
   - **Имя**: `Debian 12.12`
   - **Архитектура**: `x86_64`
   - **Media**: `Debian mirror`
   - **Partition Tables**:
     - `Simple Debian 12 (no crypto)`
     - `Simple Debian 12 LUKS` (если используется шифрование)
   - **PXE Loader**: `iPXE UEFI HTTP`

## Настройка подсети

1. Перейдите в **Инфраструктура → Подсети → Создать подсеть**
2. Заполните:
   - **Имя**: `Internal Network`
   - **Сеть**: `192.168.0.0/24`
   - **Шлюз**: `192.168.0.1`
   - **DNS серверы**: `192.168.0.6, 8.8.8.8`
   - **Smart Proxy**: Выберите ваш Smart Proxy

## Настройка Smart Proxy

1. Перейдите в **Инфраструктура → Смарт Прокси**
2. Убедитесь, что активны функции:
   - DHCP
   - TFTP
   - HTTPBoot
   - Templates
   - Logs
